# StudyWise Roadmap

## 已完成

- [x] PDF 阅读与页码同步：React-PDF 连续阅读、左右页码联动、可拖拽分栏。
- [x] 异步 OCR：PaddleOCR 整文档解析、页面 Markdown / 图片 / 渲染图本地保存。
- [x] 文档对话：多模型、LaTeX、历史会话、按需读取页面与图片。
- [x] 工具检索优化：按页读取、全文关键词搜索、跨页上下文补读与提示缓存。
- [x] Agent 活动轨迹：显示 Working / Worked 耗时、工具参数和结果摘要，并随会话保存。

## 下一步

- [ ] PDF 选区提问：选中文本、公式、图片或表格后，直接携带页码与内容提问。
- [ ] 回答来源引用：回答标记依据页码；点击引用可跳回 PDF 对应位置。
- [ ] 页面笔记与摘录：笔记、收藏公式和模型解释锚定到具体页面或选区。
- [ ] 小测验与错题：从页面或选区出题、判分、保存错题并支持后续复习。
- [ ] 文档学习地图：生成章节主题、核心概念、关键公式与建议学习顺序。
- [ ] OCR 纠错与原文对照：标记或修正 OCR 错误，让模型在必要时优先查看页面渲染图。
- [ ] 工具调用评测：用固定案例持续评估检索页码、上下文完整性、token 与耗时。

## 后续考虑

- [ ] 项目级跨文档问答：在同一 Project 的讲义、论文和作业之间检索与对比。
- [ ] RAG / 向量检索：仅在跨文档和大规模资料检索成为高频需求后引入。
- [ ] Tauri 桌面发布：将前后端打包为独立桌面应用。
